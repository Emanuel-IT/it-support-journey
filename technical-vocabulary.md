# Technical Vocabulary

This file contains technical terms I am learning during my IT Support & Cloud journey.

---

## Networking

- IP Address: A unique address that identifies a device on a network.
- DNS (Domain Name System): Translates domain names into IP addresses.
- DHCP: Automatically assigns IP addresses to devices.
- Router: Connects different networks.
- Switch: Connects devices within the same network.

---

## Hardware

- CPU: The brain of the computer that processes instructions.
- RAM: Temporary memory used while programs are running.
- SSD: Solid-state storage device (faster than HDD).
- HDD: Traditional hard disk storage.
- Motherboard: Main circuit board that connects all components.
